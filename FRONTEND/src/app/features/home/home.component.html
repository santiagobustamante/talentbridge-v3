shield

Comparte información profesional con responsabilidad

TalentBridge permite compartir enlaces como LinkedIn, GitHub o portafolios. Antes de publicar repositorios, verifica que no contengan credenciales, secretos, datos de clientes o código protegido por acuerdos de confidencialidad.
