## Warnings de build conocidos (no bloquean, no son nuevos)

`ng build` del frontend deja warnings de presupuesto de bundle (`budget.initial` y varios `*.component.scss` individuales excediendo 6kB) que ya existían antes de esta sesión. **Excepción 2026-07-17 (Fase 15):** el nuevo `styles/_forms.scss` se consume vía mixins de Sass (no clases compartidas), que duplican su CSS en cada archivo que los `@include` — esto probablemente aumentó unos cientos de bytes el `.scss` compilado de los ~9 componentes tocados (varios ya estaban por encima del presupuesto de 6kB antes del cambio). Es la misma categoría de warning no bloqueante, no una regresión funcional — ver `DECISIONS.md` para el detalle de por qué se aceptó ese costo en vez de migrar a clases compartidas. Ver el detalle completo corriendo `ng build` — la lista cambia con cada commit y no vale la pena fijarla acá como texto estático.
